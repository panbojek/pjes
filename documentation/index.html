contact form

files:

  • ajax-mail.js ==> for ajax
  • mail.php ==> for mail

customize contact form:

This Template Contact form is DYNAMIC. It works with ajax! So "Thank you" and warning message show in this page. Don't need to go another page. for customization here given example: 

// Set the recipient email address.
// Update this to your desired email address line number 23 in mail.php
$recipient = "example@domain.com";

// Send the email.
// User get warning message if all field isn't fill
// To Change this message go to line number 17 in mail.php
echo "Please complete the form and try again.";

// User get thank you message if send a message
// To Change this message go to line number 37 in mail.php
echo "Thank You! Your message has been sent.";
